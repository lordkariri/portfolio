Iterative Design

This project is a collaboration between Elise Carman, Gianna Aiello, Julia Zdzilowska, Karina Valrani, and Zest - an online gifting platform helping brands scale by streamlining the gift-buying experience directly on their website. It offers multi-ship checkout, e-gifting, and corporate storefronts, with a concierge feature for brands to manage or process orders. Our redesign transforms a fragmented, self-guided setup process into a structured, intuitive flow that reduces reliance on customer support, shortens time to completion, and improves overall usability.

Accessible Components

This project examined how popup dialogs function across major platforms—Google Drive, LinkedIn, and Instagram—by evaluating their accessibility through mouse, keyboard, touch, and screen reader inputs. Using those insights, I redesigned the popup dialog to improve usability, efficiency, and accessibility, introducing clearer focus indicators, better screen reader support, hover and pressed states, and more intuitive dismiss options. The redesign addresses real-world mismatches by making the component inclusive for users with a range of abilities and input preferences.

Capriccio

This project bridges visual art and music through code. I built an interactive system that takes an artwork as input—an image—and translates its visual elements like color, shape, and composition into a generative musical composition. It’s an experiment in synesthesia and storytelling, turning still visuals into soundscapes that echo the emotion of the original piece.
